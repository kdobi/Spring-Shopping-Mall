## 🔁개요
- Java, Spring, MySQL을 활용하여 간단한 쇼핑몰 기능을 구현하는 프로젝트
- 웹 서버와 데이터베이스를 연동하며 회원, 상품, 주문 흐름을 직접 구현해보기 위한 프로젝트
- 단순한 화면 구현을 넘어서 백엔드 로직과 데이터 흐름을 이해하는 것을 목표로 진행

## 📂커밋 컨벤션 가이드
- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 수정
- style: 코드 스타일 변경
- design: 사용자 UI 디자인 변경
- perf: 성능 개선
- chore: 자잘한 수정이나 빌드 업데이트
- rename: 파일 혹은 폴더명 수정
- remove: 파일 삭제
- test: 테스트 코드 추가 및 수정
- refactor: 리팩토링
- build: 빌드 파일 수정
- ci: CI 설정 파일 수정
<br>

## 🚩목표
